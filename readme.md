# API Automation Test Framework

### pre-requisite
```
NodeJS
Visual Studio Code
```

### clone repository
```
git clone https://github.com/shamapar/playwright-api-test-framework.git
```

### command to install package
```
npm install
```

### command to run tests
```
npx playwright test
```
